### Description
Baremetal on ARM and AVR architecture MCUs.
###### Features:
* no usage of external libraries
* no IDE
* custom Makefile
* custom linker script

#### Structure
```
BareMetal
├── arm
│   ├── c
│   │   ├── docs
│   │   ├── etc
│   │   ├── include
│   │   ├── Makefile
│   │   ├── README.md
│   │   └── source
│   └── cpp
│       ├── etc
│       ├── makefile
│       ├── README.md
│       └── source
└── avr
    ├── etc
    ├── makefile
    ├── README.md
    └── source
```
